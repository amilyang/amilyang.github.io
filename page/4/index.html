hexo 入门

受朋友影响想创建一个属于自己的技术博客，可以把平时用到的技术积累写在这里，因为网速的影响，加上不是很熟悉创建的流程，所以耗费了多半天时间终于成功了，记录一下此时此刻的心情，从今天开始，以后要每周写整理一次学习内容，发一篇博客。

创建新文章

hexo new “文章名称”

生成新页面

hexo g

开启本地服务

hexo s

清除旧数据

hexo clean


| 配置项     | 意义   |
| ---------- | ------ |
| title    | 网页文章标题      |
| date     | 文章创建日期    |
| comments   | 文章评论功能是否启动     |
| tags   | 文章标签    | 
| categories       | 文章分类    |
| keywords       | 文章关键字   |
