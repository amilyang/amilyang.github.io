两个数组的合并

合并数据的三种方法：

方法一： ES6 扩展运算符（spread）是三个点（…）, 它如同rest参数的逆运算， 将一个数组转为用逗号分隔的参数序列

var arr1 = [1,2,3,4]
var arr2 = [4,5,6,7]
arr1.push(...arr2)
console.log(arr1) // [1,2,3,4,4,5,6,7]

方法二： push 虽然没有提供处理数组的功能， 但apply会将一个数组转换成一个参数i接一个参数传递过去，可以利用a.push.apply(a,b);

var arr1 = [1,2,3,4]
var arr2 = [4,5,6,7]
arr1.push.apply(arr1, arr2) 
console.log(arr1) // [1,2,3,4,4,5,6,7]

方法三：concat

var arr1 = [1,2,3,4]
var arr2 = [5,6,7,8]
const arr = arr1.concat(arr2)
console.log(arr) // [1,2,3,4,4,5,6,7]

方法四： for循环 思路是：遍历其中一个数组，把该数组中的所有元素依次添加到另外一个数组中。

var arr1 = [1,2,3,4]
var arr2 = [4,5,6,7]
for(var value in arr2){
  arr1.push(value)
}
console.log(arr1) // [1,2,3,4,4,5,6,7]
